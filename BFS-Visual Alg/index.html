Welcome to our Breadth-First Search Visualization! To get started, follow the steps below:

  1. To start, add some nodes by clicking inside the svg box below. If you'd like to remove a node that you created, click the existing node 3 times.
  2. To add edges inbetween the nodes, first click on an existing node once to turn it pink, and then click on the other existing node you would like to connect it to once.
  3. Step 2 turns off the function that allows you to add nodes. If you would like to add more nodes, click the "Add More Nodes" button and repeat step 1 and 2.
  4. After you're done adding in your nodes and edges, you can do two things. Either (in any order):
  • click the "Show Adjacency List" button to show a table of the nodes you input and their neighbors through each edge connected, or
  • click the "Traverse BFS" multiple times to step through the breadth-first search visualization of your graph.
